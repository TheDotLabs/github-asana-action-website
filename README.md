# Github Asana Action Website

Github Asana Action Website built with SvelteJs and Tailwindcss.

Visit here for [Github Asana Action](https://github.com/TheDotLabs/github-asana-action)

<img src="./res/s1.png" alt="img"/>

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
